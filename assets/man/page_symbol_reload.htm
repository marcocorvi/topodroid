DRAWING TOOL SETS

TopoDroid comes with six sets of drawing tools:
  • speleology
  • archeology
  • geology
  • biology
  • paleonthology
  • mining
By default only the speleology tools are installed.
With the "PALETTE" menu of the main window you get a dialog to install additional tools. Check the tool sets you want to add and press the button "Install".
If you press the button "Replace" the currently installed tools are removed before installing those you selected.

After installing you must enable the tools you need, using the "Palette" button of the main window. The choice of enabled tools is stored in the database and is not lost when tools are removed.
