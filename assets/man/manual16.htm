3D sketching

3D sketching is done around one shot at a time. Therefore the display is centered about the current shot.

The 3D-sketch window has three modes (and three different button bars):

  • move
  • draw
  • select
As for the 2D sketch window, the leftmost green buttons indicate the current mode.
The other buttons for the move mode are:
  • surface display mode
  • surface create
  • data download
  • notes
  • info
Surface display mode The surface display mode can be,
  • only the surface of the current shot
  • the surfaces of the current and the nearby shots
  • all the surfaces
  • no surface.

When no suface is displayed, you can step to the nearby shots by selecting the shot.

In the draw mode you can place iconic symbols on the surface, as well as trace line symbols.

  • undo
  • redo
  • palette (item picker)
For the select mode:
  • select mode
  • select size
  • previous item (not implemented)
  • next item (not implemented)
Menus
  • export (supported formats: Therion 3D, DXF)
  • palette
  • zoom fit
  • head: orienting the model using the android device
  • delete the 3D model
  • clear: remove all the surfaces
  • settings
  • help
