AI helper dialog

This dialog allows you to ask questions to the AI and get help for TopoDroid.
Currently only Gemini models are suuported, and you need to provide your own API key through the appropriate setting.

Enter your question in the top row, and press the "Submit" button to send it to the AI.
The answer is shown in the central text box.

Note To use the AI assistant you need internet connection.

The button "Reset" resets the chat history.
The button "Clear" clears the question box.
The button "Close" closes the dialog.
