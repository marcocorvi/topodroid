AI helper dialog

This dialog allows you to ask questions to the AI and get help for TopoDroidi and assistance for the app settings.
Currently only Gemini models are supported, and you need to provide your own API key through the appropriate setting dialog.

The top row contains

  • model selector
  • model temperature (between 0 and 1: the lower, the less allucinating) [0.2]
  • chat max tokens [1024]

Enter your question in the next row, and press the "Submit" button to send it to the AI.
The answer is shown in the central scrollable text box.

The local context for help assistant is the user manual.
The local context for the settings assistant is a list with the description of the settings. This local context can be overridden by putting a text file "settings.txt", with context instructions, in the app private folder.

Notes
To use the AI assistant you need internet connection.
If you get a "MAX TOKENS" error, try with a short question or a 'lite' model.

The button "Reset" resets the chat history.
The button "Clear" clears the question box.
The button "Close" closes the dialog.

WARNING
The AI can make errors. In particular it can fail to identify the setting that is related to your request or the proper value, especially if the request is phrased in negative terms.
