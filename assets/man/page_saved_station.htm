SAVED STATIONS

This dialog is opened from the Station button in the Survey Data window, or from the edit station dialog of the Drawing window.

The "saved stations" are station names saved for later reference, eg, at a crossway, to go back and continue the survey from there, or to mark a possible lead to check in a future trip.

Note The names of stations attached to shots or to "saved stations" are just names and do not identify entities. Therefore, when a station name of a shot is changed, the same name in a "saved station" does not change.

The dialog displays the list of saved stations. You select one by tapping it.

The selected station is shown in the text fields. Opened from the Survey Data window, the text field contains the last station, or the active station if this is set. Opened from the edit-station dialog, the text field contains the station.

Each saved station must have a brief description (comment) or a flag set.
It is possible to assign a geocode to a saved station [T].

The dialog displays

  • the selected station
  • the station flag, if any ("fixed" or "painted")
  • the station comment

Buttons:
  • Clear the station fields: name comment and flag
  • Geocode sets/changes the station geocode [T]
  • Save the station into the database
  • Delete the station from the database (non recoverable)
  • Close closes the dialog
  • Active: sets the station as the "active" station, and closes the dialog
