Sketch statistics

This dialog is opened tapping the Statistics menu of the Sketch window (plan or profile view).

The survey statistics are

  • number of (regular) legs, their total, profile, and plan lengths
  • number of duplicate legs, and their total length
  • number of surface leg, and their total length
  • number of splay shots
  • number of stations
  • number of cycles (including the backshot-cycles)
  • number of disconnected surveys (this is 1 if the legs are all attached together in one survey)
The sketch related statistics are
  • origin station (zero station)
  • the survey length
  • 3D length
  • length of the plan-projection
  • the survey extension (West, East, North and South)
  • the survey depths, both positive (above the zero station) and negative (below the zero station)
  • the number of stations
  • the number of shots (regular, duplicate, and surface)
  • the number of splays
  • average angle error between shots of the same leg [degrees]
  • the number of loops

For each loop:

  • closing shot
  • closing error: 3d length over total loop length
  • horizontal and vertical error lengths
  • parcent error: ratio between error and loop length
  • average angular error [degrres]

Note: length are in the units set in the app settings.
