SKETCH STATISTICS

This dialog is opened from the Stats menu of the Drawing window (plan or profile view). It displays both data-reduction statistics and sketch related statistics.

The data-reduction statistics are

  • Legs: number of (regular) legs, sketch-leg total length, profile and plan view cave-leg total lengths.
  • Duplicate shots: number of duplicate legs and their total length.
  • Surface shots: number of surface leg and their total length.
  • Splays: number of splay shots, and sketch splay shots.
  • Stations: number of stations, and sketch stations.
  • Leg sigma: average position error between repeated shots.
  • Survey length: length of legs not including flagged or repeated legs.
  • Dangling shots: number of shots not attached to the sketch if any, and their total length. Tapping this line opens the dialog with the list of unattached shots.
  • Cycles: number of cycles (including the backshot-cycles) if any. This item is only shown if there is at least one loop defined.
  • Components: number of disconnected survey components if greater than 1. This is 1 if the legs are all attached together in one survey.
  • Sketch origin station: sketch origin (zero station).
  • Sketch length: the midline length.
  • 3D length: profile-view (3D) midline length.
  • Plan length: plan-view midline length.
  • W E N S: the sketch extension (West, East, North and South).
  • Sketch denivelation: the sketch denivelations, both positive and negative, relative to the sketch center (the sketch origin unless re-centered).
  • Cave denivelation: the cave denivelation relative to the sketch origin.
  • Angle errors: average angle error between shots of the same leg [degrees].
  • the number of loops, those used in closure compensation and those not used

When there are two values separated by a number sign (#), the first value is the total survey value, and the second is the data reduction value, i.e., the value for the connected component that contains the reference station of sketch. These two values are different when the survey midline data contains more than one connected component.

For each loop:
  • closing shot
  • number of loop shots
  • closing error: 3d length over total loop length
  • horizontal and vertical error lengths
  • percent error: ratio between error and loop length
  • estimated shot angle error [degrees]. This is the error times the square root of the number of shots and divided by the length. The result, in radians, is converted to degrees.

Tapping on the loop closure entry switches the text between the closure error, and the list of loop stations.

Length are in the units set in the app settings.

Denivelations
The sketch center is the sketch origin, unless the sketch is re-centered.

The cave denivelation includes the survey data that are not "surface", "duplicate", or "commented". It includes the splays, therefore if you have taken "surface" splays you need to mark them as "surface".
