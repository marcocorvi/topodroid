Sketch list dialog

The "Sketch" button ("picture" icon) in the "Shot list" window opens the "Sketch list" dialog.

This dialog has a button to create a new sketch and the list of the existing sketches of the survey.

New sketch creates a new sketch. It takes you to another dialog, where you can enter the sketch information, before opening the "Sketch" window, where you can draw and edit the sketch.

The list of the existing sketches has two columns:
plan views on the left
profile views on the right

Each sketch consists of a plan and a profile view, and it is listed in both columns.

Tap on one sketch entry to open it. If you tap the "plan" entry the sketch is opened in the plan view. If you tap the "profile" entry the sketch is opened in the profile view.
