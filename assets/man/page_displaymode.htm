SURVEY DATA DISPLAY MODE

This dialog is opened from the Reference button in the Survey Data window.

  • shot id
  • splay: shots with only one station
  • latest splays: only available if Secondary settings > Survey data > Recent shots setting is enabled. Time limit defined by Secondary settings > Survey data > Recent data timeout setting (in seconds).
  • blank: shots with no stations
  • repeated leg: shots close to the preceding leg

The shot id's are unique increasing indices assigned by TopoDroid to the shots of the survey.

If the splay shots are hidden it is still possible to show the splay shots for individual stations. Tapping on the station name in a leg shot opens the Saved stations dialog through which the display of the splay a the specified station can betoggled on and off.
