3D VIEWER - Surveys list

When you open the 3D of a cave project, a long press on the seventh button ("fat pencil") opens a dialog with the list of the surveys in the project.
For each survey it shows
  • color box
  • visibility box
  • name of the survey
The color box allows to change the color assigned to the survey, through a color picker dialog.

The visibility box toggles the visibility of the survey in the 3D.

Buttons

  • OK
  • Cancel
