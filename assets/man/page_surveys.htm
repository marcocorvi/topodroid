Surveys bulk operations

This dialog is opend from the Main Window menu "BULK OPS."

It allows to do bulk operation on a set of selected surveys.

The available operations are

  • delete: delete the surevys. Thers is confirmation question
  • export: export the surveys. The export format can be choosen afterward
  • import: import all zip archives from a folder

Other buttons
  • toggle: toggle the surveys selection
  • cancel: close the dialog
