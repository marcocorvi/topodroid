3D model export

The 3D export dialog is opened by the menu "Export" in the 3D viewer window.

The 3D model can be exported as

  • STL, either binary or ascii
  • KML (if the model is georeferenced)
  • CGAL
  • DXF
  • Shapefile
  • LAS, binary
  • glTF (GL transfer format)

When supported by the format, the exported file can include

  • Stations
  • Splays
  • Walls (if generated)
  • Surface DEM (if loaded)
