Photo edit

This dialog is opened tapping an entry in the Photo list dialog.

  • thumbnail of the photo
  • orientation (azimuth and clino) of the photo
  • date and time of the photo
  • comment (editable)
  • Save saves the changes to the comment
  • Delete deletes the photo (if the photo is for a shot)

The image is displayed when you tap on the thumbnail. The back button closes it.

Note
Photo taken for sketch points can be viewed also through the item edit dialog, and are deleted when the photo point is deleted.
