Shot edit (1)

This dialog is opened by tapping on the shot data in the Shot list window.

It displays

  • length, azimuth, inclination. Editable for manually entered shots; not for downloaded shots
  • differences from the average acceleration, magnetic field, and magnetic dip. Only for DistoX2 shots
  • stations
  • left arrow: load previous leg
  • reverse: swap stations
  • right arrow: load next leg
  • shot comment
  • profile "extend" choice: "left", "vertical", "right"
  • six check buttons:

    duplicate and surface: shot flags
    Previous leg: the shot belongs to the immediately preceeding leg.
    Next leg: the shot belongs to the immediately following leg.
    Renumber: renumber shots from this leg onward following the current station naming policy. Both stations must have been specified
    Splay group: apply the station change to all splays in the group of the splay shot. It applies only to the splays with same station as the edit shot. Only FROM station must have been specified
    X-Splay toggle X-splay marks for this shot and all previous splays of the group. X-splays are shown in green.

  • OK: save the changes and close the dialog
  • Save: save the changes without closing the dialog
  • More: switch to the secondary edit dialog
  • Cancel: close the dialog without saving

Station cut-n-paste
Station names can be 'cut', 'copied', and 'pasted' across the app, in any dialog where there is a station entry field.

Long-tapping the station entry field opens the drop-down menu with the three options

  • cut
  • copy
  • paste
