TopoDroid settings

GENERAL SETTINGS
  • Work directory. Default "unset" (ie "TopoDroid")
  • Text size >= 1. Default 14
  • Buttons size {small, big}. Default small
  • Activity level {basic,normal,advanced,experimental}. Default normal
  • Cosurveying service. Default NO
  • Language {en,es,fr,de,it,ru,bg}. Default "unset" (i.e. locale)

SURVEY SETTINGS

  • Team. Default "unset"
  • Survey stations naming policy {none, splay+fore, splay+back, fore+splay, back+splay, back+splay+fore}. Default "splay+fore".
  • Station names {alphanumeric,numeric}. Default alphanumeric
  • Initial station name. Default "unset"
  • Photo thumbnails size [80,400]. Default 200 pxl
UNITS
  • Length units {m,ft}. Default m
  • Angle units {deg,grad}. Default deg
  • Sketch grid units {m,yd}. Default m
SHOTS
  • Leg shot closeness > 0 [percent]. Default 0.05 (5%)
  • Minimum number of shots per leg >=2. Default 2
  • Limit for splay extend "vertical" [0,90]. Default 10 deg
  • Leg V threshold [0,90]. Default 80 deg
  • Settable "extend" reference. Default NO
  • Loop closure. Default NO
  • Backsight in user entered shot dialog. Default NO
  • Magnetic anomaly compensation. Default NO
  • Shot direction delay (timer). Default 10 s
  • Shot direction timer volume [10,100]. Default 50 percent
LOCATION
  • Location units {ddmmss, dec.deg}. Default ddmmss
ACCURACY
  • Accel. threshold > 0. Default 400
  • Magn. threshold > 0. Default 300
  • Dip threshold > 0. Default 3 deg

SKETCH SETTINGS

  • Drawing tool picker {recent,list,grid}. Default recent
  • Recent tool number {3, 4, 5, 6}. Default 4
  • Side drag. Default NO
  • Zoom controls {off,temporary,permanent}. Default permanent
  • Check dangling shots. Default NO
  • Save areas in xtherion-"style". Default NO
POINT TOOLS
  • Non-zoomed point icons. Default NO
  • Point tools scale. Default 1.2
  • Labels size. Default 24 px
LINE TOOLS
  • Lines width (N.B. wall line are twice this). Default 1 px
  • Line style {Bezier,fine,normal,coarse}. Default normal
  • Minimum distance between line points. Default 10 px
  • Section lines "direction tick" size [1,20]. Default 8 [x unit]
  • Area border visibility. Default YES
  • Continuation {join,touch}. Default touch
  • Bezier line accuracy. Default 1
  • Bezier line corner threshold. Default 20
SKETCH ITEMS
  • Survey lines width [0.5,10]. Default 1 px
  • Station names size. Default 24 px
  • Green dots radius [0.5,100]. Default 5 px
  • Selection radius >= 0. Default 24
  • Splay plan-view threshold [0,90]. Default 80 deg
  • Dashed splay clino threshold [0,90]. Default 50 deg
  • Cross-section H threshold [0,90]. Default 70 deg
AUTO-WALLS
  • Walls type {none,convex}. Default none
  • Splay plan clino. Default 70 deg
  • Splay profile clino. Default 45 deg
  • Separation. Default 0.1 m
  • Concavity. Default 0.1
  • Step. Default 1 m

IMPORT/EXPORT SETTINGS

  • PocketTopo color map
  • Estimate LR extend on import, Default YES
  • Default data export type {th,svx,dat,srv,top,tro,dxf,svg,csv} default th
  • Add stations to Therion scraps. Default YES
  • cSurvey station prefix. Default NO
  • Survex line termination {Linux,Windows}. Default Linux
  • Survex splays TO station. Default NO
  • Survex LRUD. Default NO
  • CSV length units {m,ft}. Default meters
  • PNG resolution [0.5, 10]. Default 1.5 [x200 px]
  • PNG background color (RGB in 0-100). Default "0 0 0" [black]
  • DXF sketch scale. Default 1
  • AutoCAD version {6,12}. Default 6

DEVICE SETTINGS

  • Check if BT is enabled. Default YES
  • Connection data mode {on-demand,continuous}. Default on-demand
  • Auto reconnect (in continuous mode). Default FALSE
  • BT socket type {normal,insecure}. Default normal
  • Z6 workaround. Default YES
  • Connection setup retries >= 0. Default 0
  • Autopairing. Default YES

CALIBRATION SETTINGS

  • Group policy {4x4+M,Nx4,distance). Default Nx4
  • Group distance threshold. Default 40 [deg]
  • Calibration epsilon. Default 0.000001
  • Calibration maximum iterations. Default 200
  • Show calibration raw data. Default NO
  • Calibration algorithm {auto,linear,non-linear}. Default auto
