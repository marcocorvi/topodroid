CALIBRATION DATA

This dialog is opened tapping a data entry in the Calibration Data window.

  • Shot ID (index)
  • azimuth, inclination, roll
  • calibration error
  • group number entry field
  • Save the changes

The buttons are

  • mark the data "deleted"
  • reassign the groups from this data onward

The item group can be set or changed.

  • positive: the data takes part to the computation of the calibration coefficients
  • zero: the data does not enter the computation
  • negative: the data does not take part in the group assignment, nor in the computation of the calibration coefficients
Data with the same positive group number form a group.

Deleted data are not removed from the database.
They are not taken into account in the automatic group assignment, and do not enter to the computation of the calibration coefficients.
They are not usually shown in the calibration data list. If displayed (see Calibration Data window) they have a gray background.

The group number assignment can follow the TopoDroid scheme or PocketTopo scheme [T].
