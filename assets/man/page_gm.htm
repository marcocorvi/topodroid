DistoX CALIBRATION DATA

This dialog is opened tapping a data entry in the calibration data window.

  • azimuth, inclination, roll
  • calibration error
  • group number entry field
  • Save the changes

The buttons are

  • reassign the groups from this data onward
  • mark the data "deleted"

The item group can be set or changed.

  • positive: the data takes part to the computation of the calibration coefficients
  • zero: the data does not enter the computation
Data with the same positive group number form a group.

Deleted data are not removed from the database.
They are not usually shown in the calibration data list, are not taken into account in the automatic group assignment, and do not enter to the computation of the calibration coefficients.
