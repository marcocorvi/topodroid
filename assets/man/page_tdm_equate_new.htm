PROJECT MANAGER - New equate dialog

This dialog is opened by the New equate button on the Project Display window

You can manually add an equate between stations of the project surveys by entering the station names of the displayed surveys. The "-" (dash) sign excludes the given measurement from the relationship.

The names of the stations are shown in the plan view.

Buttons:

  • Cancel: closes the dialog
  • Search: finds the next points with the same name in the measurements
  • All: creates a equate between all points with the same name
  • OK: confirms the changes and closes the dialog.
