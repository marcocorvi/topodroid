TopoDroid HELP

The help-dialog is opened from the Help menu of the TopoDroid windows.

It contains a brief explanation of the window buttons and menus.

The book button at the top right opens the user manual at the section about the current window. The user manual page can be opened also with the Android hardware MENU button.

A long tap on the book button opens the user manual at the first section.

Gemini AI key
In order to use AI hep you need to have your own API key and store it in the settings. Think of this as a private "password" that allow TopoDroid to ask Gemini questions on your behalf.
The API key is never exported when you export the settings.

To get your API key

  1. sign in to Google AI Studio: go to aistudio,google.com and sign in with your Google/Gmail account. If prompted review and accept the Terms of Services
  2. create your key: on the left sidebar click on "Get API key", click on the button "Create API key in new project": the long string of letters and number that appers is your API key
  3. copy and save this key. Keep it private. Anyone who has it can use your allowance of questions
  4. paste the key in the TopoDroid AI API key setting
Note.
There is a limit to the number of questions per day that you can ask.
It does not work underground, but you can use the AI to improive your TopoDroid skill while at the camp or at home.
