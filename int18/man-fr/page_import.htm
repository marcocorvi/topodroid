IMPORTATION D'UN RELEVÉ TOPOGRAPHIQUE

Cette boîte de dialogue s'ouvre en appuyant sur le bouton Import fichier de la fenêtre principale.

Elle affiche la liste des formats de fichiers d'importation pris en charge :

  • Therion : fichiers de données ".th", avec un format de données simple ("normal, de, vers, longueur, pente" et optionnellement GDHB). Les brouillons de Therion ne sont pas importés.
  • Compass : fichiers ".dat" de Compass.
  • VisualTopo : fichiers ".tro" et "trox"
  • PocketTopo : fichiers ".top" . Les données et les dessins sont importés.
  • Archives ZIP de TopoDroid. Ce sont des fichiers zip spéciaux contenant toutes les informations pour alimenter la base de données, les dessins (fichiers ".tdr"), le fichier d'annotation et les photos.

Le fichier d'importation est sélectionné avec le fournisseur de contenu du système.

Pour certains formats, il est possible de personnaliser l'importation grâce à quelques marqueurs.

Options d'importation de Compass (dat)

  • LRUD (GDHB) si activé les valeurs LRUD sont importées et converties en visées d'habillage.
  • Leg first (cheminement en premier) si activé, les visées latérales LRUD sont placées après le cheminement.
  • Diving datamode (mode plongée) valide le mode plongée des données dans le fichier Compass [T].


Options d'importation de VisualTopo (tro, trox)

  • LRUD si activé les valeurs LRUD sont importées et converties en visées d'habillage.
  • Leg first (cheminement en premier) si activé, les visées latérales LRUD sont placées après le cheminement.

N.B. L'importation peut échouer (et faire planter l'application) en raison d'une défaillance de la base de données que TopoDroid ne peut pas intercepter. Si cela se produit, supprimez le relevé partiellement importé et réessayez de l'importer à nouveau.
