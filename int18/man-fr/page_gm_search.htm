Calib Data SEARCH

The dialog is opened by the Search button of the Calib Data window.

It allows o search for calibration data with error exceeding a given minimum error, which can by typed in the text field.
The calibration data with error value larger than the minimum error are highlighted with an orange background to the error value.

There are three buttons:

  • Close the dialog without doing anything.
  • Cancel the display of the search result.
  • Search perform the search and display the result.
