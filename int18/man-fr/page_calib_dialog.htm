ÉTALONNAGES du DistoX

Cette boîte de dialogue s'ouvre en appuyant sur le bouton d'étalonnage de la fenêtre de l'appareil périphérique.

Elle dispose de boutons pour :

  • Nouvel étalonnage
  • Importer un étalonnage depuis un export CSV
  • Réinitialisation d'usine des coefficients d'étalonnage du DistoX

Au-dessous il y a la liste des différents étalonnages de l'appareil actif.

Ouvrir un étalonnage

Dans la liste, appuyez sur le nom de l’étalonnage voulu afin de l'ouvrir dans la Fenêtre d'étalonnage.

Nouvel étalonnage ouvre la Fenêtre d'étalonnage, où vous pouvez entrer les données afin de créer un nouvel étalonnage. Après qu'il ait été enregistré dans la base de données vous pourrez l'ouvrir (dans la fenêtre des Données d'étalonnage), télécharger les données d'étalonnage, calculer les coefficients d'étalonnage et les charger sur le DistoX.
