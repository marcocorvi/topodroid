OUTILS LIGNES

Les outils "ligne" sont définis dans des fichiers texte.

Deux entrées spécifiques : "user" et "paroi", sont intégrées dans le programme. Les autres fichiers "lignes" sont téléchargés à partir du sous-répertoire symbol/line de TopoDroid. Ce répertoire est rempli avec les outils de points compilés dans l'apk de distribution. Vous pouvez supprimer les outils dont vous n'avez pas besoin, modifier ceux que vous souhaitez modifier et en ajouter de nouveaux qui ne sont pas fournis dans l'apk.


Le fichier d'outil de ligne contient des lignes spécifiant la propriété de l'outil de dessin.

  • "symbol line" [obligatoire]
  • "name NAME", where NAME is the area default name [obligatoire]
  • translated names, eg, "name-es ..." [optionnel]
  • "th_name NAME", where NAME is the Therion name, and must be the same as the filename [obligatoire]
  • "group GROUP", specifies the group this line belongs to [optionnel, default no group]
  • "color rgb alpha" (rgb and alpha in hex format) [rgb obligatoire, alpha is optionnel]
  • "width SIZE" [optionnel, multiple of "base line width"]
  • "dash PATTERN" [optionnel dash pattern]
  • "style STYLE" where STYLE can be 'straight' or 'xN'
  • "effect" begins the line path-effect and is followed by path commands, and closed by
  • "endeffect"
  • "roundtrip VALUE" [optionnel, pour SVG-roundtrip, VALUE peut être 1 (walls_shp), 2 (walls_sym), 3 (detail_shp), 4 (detail_sym) - default 3]
  • "level" marqueur de vue [optionnel, 1 si manquant]
  • "endsymbol"

Nom
Reportez-vous à l'outil POINTS pour savoir comment spécifier les noms.

Groupe
Cet attribut est utilisé pour remplacer un symbole manquant par le symbole du groupe.

Tiret
La propriété "dash" spécifie que la ligne est dessinée avec un schéma en pointillés. Par exemple, "tiret 15 5" signifie que la ligne est dessinée avec des segments de 15 unités, séparés par des espaces de 5 unités.

Style
La propriété "style" permet de dessiner des lignes en utilisant des points épars.
Un "style x3" signifie que la ligne est dessinée en utilisant un point sur trois. L'effet est que la ligne est moins "courbe".
Si vous voulez des segments droits, utilisez "style straight".

Effet de tracé (Effect path)
Cette option est utilisée pour définir des lignes avec un style complexe.

Niveau
L'attribut "level" prend la valeur de l'indicateur des vues activées sur ce poste.
L'indicateur est un nombre composé de 1 (base), 2 (plancher), 4 (remplissages), 8 (plafond) et 16 (artefacts).
